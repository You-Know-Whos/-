# 这是一个沙盒小游戏

## 操作指南

### 创造模式：
* **鼠标左键**：放置物体
* **鼠标右键**：摧毁物体
* **鼠标滚轮**：切换物体
* **上方向键/下方向键**：物体放置距离+/-

### 自由模式：
* **鼠标左键（按住）**：拖拽物体
* **鼠标右键**：开关特殊物体的功能

### 任何模式：
* **1**：切换创造模式
* **2**：切换自由模式
* **W/S/A/D**：控制镜头移动
* **Q/E**：控制镜头旋转
* **Z（+ Shift）**：控制镜头缩放
* **X**：切换重力方向
* **空格**：切换暂停/开始
* **Tab**：切换时间流速（快/慢）
